# 🏧 ATM System Management

## Turning Java Concepts into a Real-World Banking Experience

This project is a console-based ATM Management System developed using Java 17. It was created to apply core Java concepts in a practical and meaningful way by simulating everyday banking operations.

Rather than focusing only on theory, this project demonstrates how programming concepts can be used to build a structured and interactive application. Users can create accounts, manage transactions, check balances, and interact with the system through a simple menu-driven interface.

The project reflects my journey of learning Java by building something that resembles a real-world banking environment while strengthening my understanding of object-oriented programming and application design.

---

## ✨ Highlights

* Create and manage customer accounts
* Deposit and withdraw funds
* View account balances

* Interactive menu-based navigation
